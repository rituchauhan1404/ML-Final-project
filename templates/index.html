About

Two datasets, namely "CARS" and "SATELLITE" has been selected to train 2 classification and 2 regression model, one for each datasets.

Following are the steps taken to prepare the models:

  • Pre-processing has been done on each of the data sets and in both the datasets, new features have been created using the existing features. Pre-processing included resolving datatype issue, finding and fixing missing and ambigous valies, converting non numeric data into numeric data via encoding, etc.
  • The clean data has been later stored in sqlite3 database in the directry named "instance"
  • Data is fetched from database and 3 classification and 3 regression model have been selected, all trained and tested, finally selecting the one with the best score among others.

Source of databases:
